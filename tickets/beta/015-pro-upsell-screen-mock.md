## 概要

Pro（ふっきん Plus）の価値訴求画面をモックとして実装し、課金なしでデザイン・コピーのユーザーテストを行う。

## スコープ

### 含む

- Pro 誘導画面（SC 相当）の UI 実装
  - 特典一覧：全9種、広告非表示、マイルーティン、クラウド同期、詳細レポート、ストリークフリーズ
  - 価格表示はプレースホルダ（「Coming Soon」等）
  - 「購入」「復元」ボタンはタップ時に「正式版で利用可能」等のメッセージ
- ホーム種目カルーセルのロック種目・設定タブ等からの遷移導線

### 含まない

- StoreKit / Play Billing 連携
- 実際のサブスク購入・復元
- A/B テスト

## 受け入れ条件

- [ ] Pro 特典が要件定義（§14.1）と一致して表示される
- [ ] ロック種目から誘導画面へ遷移できる
- [ ] 購入ボタンは課金せず、適切なメッセージを表示する
- [ ] β テスターアンケートで訴求力・分かりやすさのフィードバックを収集した

## 備考

正式版（v1.0）の課金実装チケットの UI ベースになる。
