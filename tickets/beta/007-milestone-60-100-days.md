# [BETA-007] マイルストーン 60日・100日

## メタ情報

| 項目 | 値 |
|------|-----|
| フェーズ | β |
| 要件ID | PL-12 |
| 優先度 | Should |
| 依存 | MVP 完了 |
| 参照 | `02-youken.md` §3.1.2 / `03-phase.md` §3.1 |

## 概要

MVP で実装済みのマイルストーン（3 / 7 / 14 / 30日）に加え、60日・100日の称号と特別演出を追加する。

## 追加マイルストーン

| 日数 | 称号 |
|------|------|
| 60 | 習慣の達人 |
| 100 | レジェンド |

## スコープ

### 含む

- `MilestoneService` への 60 / 100 日判定追加
- 結果画面（SC-08）での特別演出
- 記録タブ（SC-04）の称号一覧更新
- EXP 直接付与はしない（Pay to Win 回避）

### 含まない

- 60 / 100 日到達のプッシュ通知（任意）

## 受け入れ条件

- [ ] ストリーク 60日・100日到達時に特別演出が発火する
- [ ] 称号が記録タブ（カレンダー・称号一覧）に表示・永続化される
- [ ] マイルストーン達成データが再起動後も保持される

## 備考

β では到達ユーザーが少ないため、デバッグ用の日数シミュレーション手段があると検証しやすい。
