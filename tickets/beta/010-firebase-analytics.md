## 概要

継続率・ストリーク到達率など β の検証 KPI を計測するため、Firebase Analytics を本格導入する。**同意未取得時は SDK を初期化せず、イベントを送信しない**（GDPR 準拠）。

## スコープ

### 含む

- Firebase プロジェクト設定（iOS / Android）
- `firebase_analytics` パッケージ統合
- 同意状態に応じた Analytics の有効/無効切り替え
- 基本的な画面遷移・セッション計測

### 含まない

- Crashlytics（任意。余力があれば別チケット化）
- 広告関連 SDK

## 受け入れ条件

- [ ] 分析同意 ON のユーザーでのみイベントが Firebase に届く
- [ ] 同意 OFF / 未設定時はイベントが送信されない
- [ ] 同意を設定から変更した際に即時反映される
- [ ] DebugView でイベント確認ができる

## 実装メモ

- BETA-013 の同意フロー完了後に本番相当の計測を開始
- MVP では個人情報を収集しない方針。端末識別子は同意後のみ
