# [MVP-010] プランクセット（固定3種連続実行）

## メタ情報

| 項目 | 値 |
|------|-----|
| フェーズ | MVP |
| 要件ID | PL-28 |
| 優先度 | Must |
| 依存 | 002（PK-03 ハイプランク）, 008（同日複数回）, **011**（プランク実行 UI） |
| 参照 | `02-youken.md` §3.3.3 / `04-basic-design.md` SC-07b |

## 概要

ホームの「種目を選ぶ」に **プランクセット** を追加する。MVP では固定セット **SET-01**（ベーシックプランク → ニープランク → ハイプランク）を、共通の目標秒数で **3種連続** 実行できるようにする。

## セット定義（MVP）

| 項目 | 値 |
|------|-----|
| ID | `SET-01` |
| 名称 | プランクセット |
| 構成 | PK-01 → PK-02 → PK-03 |
| 秒数 | ホームスライダー1つを3種目共通 |

## スコープ

### 含む

- 種目カルーセルに **プランクセット（SET-01）** を追加（単種目と同列でスワイプ選択）
- SET-01 選択時：3種の並びプレビュー表示
- `▷ スタート` → **連続実行フロー**（SC-07b）
  - 1/3 種目タイマー → 完了 → 「次の種目へ」→ 2/3 … → 3/3 完了
- 種目ごとに `CompletePlankUseCase` 相当で EXP 計算・`WorkoutRecord` 保存
- `sessionIndexOfDay` はセット内の完了順にインクリメント（008 の再実施ボーナス・日次上限に従う）
- ストリーク +1 は **セット内の初回完了時のみ**（同日ルール維持）
- 全3種完了後、**セット結果画面**（各種目 EXP ＋ 合計 EXP）→ ホーム

### 含まない

- セット内容のユーザー編集
- ユーザー自作セットの保存（**MVP-012** / PL-30）。種目間休憩・クラウドは Pro（PL-22 / PL-23）
- 種目間の休憩タイマー（MVP は「次へ」ボタンのみで可。自動カウントダウンは任意）

## 受け入れ条件

- [ ] 種目ビューでプランクセット（SET-01）を選択できる
- [ ] スタートで PK-01 → PK-02 → PK-03 が順に実行できる
- [ ] 各種目完了ごとに EXP が付与され履歴に保存される
- [ ] 3種完了後に合算結果が表示され、ホームの進捗バーが更新される
- [ ] セット途中中断時は確認ダイアログのうえホームへ戻れる（完了分のみ保存）
- [ ] 同日2セット目以降は再実施ボーナス・日次上限が正しく適用される

## 実装メモ

- マスタ: `assets/master/plank_sets.json` または定数クラス `PlankSetDefaults`
- ユースケース: `CompletePlankSetUseCase` または実行画面内で `CompletePlankUseCase` をループ
- 画面: `PlankSetSessionScreen`（内部で種目インデックス管理。**各種目で 011 と同一 UX**）
- 結果: `PlankSetResult`（`List<PlankResult>` または合算 `PlankResult`）

## 備考

公式固定セット **SET-01** として位置づける。ユーザー自作の複数セット保存は **MVP-012**（PL-30）。Pro（PL-22/23）は休憩カスタム・クラウド同期。
