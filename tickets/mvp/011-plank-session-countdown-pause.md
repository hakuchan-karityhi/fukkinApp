## 概要

プランク実行画面（SC-07）に遷移したら、**3・2・1 の開始カウントダウン** のあと目標秒数タイマーを **自動開始** する。計測中は **一時停止ボタンのみ** を表示し、一時停止中は **再開・終了** ボタンを表示する。

## UI フロー

| 状態 | 表示 | ボタン |
|------|------|--------|
| 開始カウント | 3 → 2 → 1（1 秒間隔） | なし |
| 計測中 | 残り秒数・種目ポーズ | **一時停止** のみ |
| 一時停止中 | 残り秒数（停止中） | **再開**・**終了** |
| 完了処理中 | ローディング | なし |

## スコープ

### 含む

- `PlankSessionScreen` の状態機械: `preparing` → `running` → `paused` → …
- 開始カウント 3・2・1（`Timer` または `AnimationController`）
- カウント終了後、**ユーザー操作なし** で目標秒数タイマーを開始
- 計測中: **一時停止** ボタンのみ（「開始」「完了」ボタンを削除）
- 一時停止中: **再開**（`running` に戻る）・**終了**（確認ダイアログ → ホーム、0 EXP）
- 残り 0 秒で **自動完了** → `CompletePlankUseCase` → SC-08
- バックグラウンド復帰時の経過時間補正（既存 `PlankTimer` 方針に合わせる）

### 含まない

- 開始カウント秒数のユーザー設定（固定 3 秒）
- SC-07b プランクセット本体（010）。ただし **010 実装時は本 UX を各種目に適用** すること

## 受け入れ条件

- [ ] SC-07 遷移直後に 3・2・1 が表示され、1 秒間隔でカウントダウンする
- [ ] 1 のあと、目標秒数タイマーが自動で開始する（「開始」ボタン不要）
- [ ] 計測中は一時停止ボタンのみ表示される
- [ ] 一時停止を押すと再開・終了ボタンが表示される
- [ ] 再開で残り秒数から計測が再開し、再び一時停止のみ表示になる
- [ ] 終了で確認後ホームに戻り、EXP は付与されない
- [ ] 0 秒到達で自動的に結果画面（SC-08）へ遷移する
- [ ] `flutter analyze` / 関連テストが通過する

## 実装メモ

- 状態 enum: `PlankSessionPhase { preparing, running, paused, completing }`
- 開始カウントと本番タイマーは別 `Timer` で管理
- UI: `preparing` 時は大きな数字表示、ボタン領域は非表示
- 010 `PlankSetSessionScreen` でも同コンポーネントまたは mixin を再利用

## 備考

旧 UI（開始・完了ボタン併存）は本チケットで **置き換え** する。
