# [BETA-005] 新種目キャラポーズ・アセット追加

## メタ情報

| 項目 | 値 |
|------|-----|
| フェーズ | MVP |
| 要件ID | PL-06, PL-18 |
| 優先度 | Must |
| 依存 | BETA-002 |
| 参照 | `03-phase.md` §4.3 / `04-basic-design.md` §1.1 |

## 概要

追加プランク種目（PK-03〜09）に対応するキャラの同期プランクポーズを制作・実装する。MVP の静止画＋簡易アニメ方式を踏襲する。

## スコープ

### 含む

- 各新種目用のキャラポーズアセット（Lottie または静止画スプライト）
- プランク実行画面（SC-07）での種目連動表示
- ホーム種目ビュー（SC-03）カルーセル内の種目イラスト（モック: カンガルーのプランクポーズ）
- `poseAssetId` とマスタデータの紐づけ
- 腹筋ステージ（0〜4）との組み合わせは MVP 方式を継続（差分が大きい場合はステージ0〜2 を優先）

### 含まない

- 全ステージ × 全9種の完全差分（工数超過時は β では簡易版で可）
- セリフ拡充（BETA-006）

## 受け入れ条件

- [ ] 追加した各種目で、実行中にキャラが対応ポーズを表示する
- [ ] 種目切り替え時にポーズが正しく変わる
- [ ] アセットがアプリサイズ目安（50MB 以下）の範囲内

## 実装メモ

- MVP は2種・静止画中心。β で段階追加（`03-phase.md` §8 リスク対策）
- 種目追加バッチ（002〜004）と並行してアセット制作を進める

## 備考

工数が足りない場合は Phase1 の3種のみ先行リリースし、残りは β 中盤で追加してもよい。
